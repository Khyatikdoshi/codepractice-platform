# CodePractice — Online Coding Platform

A full-stack coding practice platform where users can solve problems, run code in the browser, and compete on a leaderboard.

## Features
- 3+ coding problems with difficulty ratings
- Monaco Editor (same engine as VS Code) with syntax highlighting
- Run JavaScript and Python code directly in the browser (no API key needed)
- Real-time leaderboard tracking solved problems per user
- Fully responsive dark-mode UI

## Tech Stack
- **Backend:** Node.js, Express.js, REST API
- **Frontend:** HTML, CSS, Vanilla JavaScript
- **Editor:** Monaco Editor
- **Python Runtime:** Pyodide (WebAssembly)
- **Storage:** JSON file-based persistence

## Run Locally
```bash
git clone https://github.com/YOUR_USERNAME/codepractice-platform.git
cd codepractice-platform
npm install
node server.js
```
Open http://localhost:3000
